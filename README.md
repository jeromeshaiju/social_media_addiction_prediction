# 📱 Social Media Addiction Predictor

This project uses Machine Learning to estimate a user's social media addiction level based on their daily habits, productivity, and sleep metrics.

## 🚀 Overview
The predictor is built using a **Decision Tree Regressor** trained on a dataset of 6,000 entries. It analyzes the correlation between digital usage patterns and focus levels to provide a categorical addiction risk assessment.

## 🛠️ Features
- **Data-Driven Analysis**: Processes 8 distinct features including screen time, study hours, and notification frequency.
- **Categorical Mapping**:
  - **0**: Low Addiction
  - **50**: Medium Addiction
  - **100**: High Addiction
- **Interactive Terminal**: User-friendly input prompts for real-time predictions.
